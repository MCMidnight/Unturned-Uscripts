# Unturned-Uscripts
## **Unturned RP Uscripts**

#### **Admin/Staff UScripts**

###### **SFine**

Command(s)
- /SFine (PlayerName) (amount) (reason)
Permission(s)
- staff.fine
  
###### **Refund**

Command(s)
- /Refund (PlayerName) (amount) (reason)

Permission(s)
- staff.refund
  
###### **Whois**

Command(s)
- /Whois

Permission(s)
- whois
  
#### **Police UScripts**

###### **Fine**

Command(s)
- /Fine (PlayerName) (amount) (reason)

Permission(s)
- pd.fine

###### **(Un)cuff**

Command(s)
- /cuff {targetPlayerName}
- /uncuff {targetPlayerName}
  
Permission(s)
- cuff
- uncuff
  
###### **Raid Warrant**

Command(s)
- /Raid (TargetPlayer) (Location) (Timer(In Minutes))

Permission(s)
- Raid

###### **Search Warrant**

Command(s)
- /Search (TargetPlayer) (Location) (Timer(In Minutes))

Permission(s)
- Search

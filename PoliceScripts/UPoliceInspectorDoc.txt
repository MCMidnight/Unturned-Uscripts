 **__Features:__**
 - Made as a replacement of Police Inspector which was on Imperial Plugins (That's why name is called PoliceInspector I just added U because of Uscript)
 - Cuff/Uncuff Players
 - Inspect - Opens storages
 - Door - Opens/Closes Doors
 - Salvage - Picks up barricade (soon to work on structures)
 - PDAnnounce - Allows Police to Announce Search and Raid Warrants
 - Simjim - Allows Police to take vechicles forcefully
 - Taser - Allows Police to tase people so they can arrest them (Easy configured (TaserID, TaserTime))
 - Panic Button - Allows Police to let others know where they died and not break NLR (Easy configured to be turned off and on)
 - 911 or 999 - Allows People to call Police Officers to there location with a reason
 - Fine - Allows Police to fine players and if configured they can get some money too (Easy configured (Payout true or false))
 
 **__Commands__**
 - /salvage - Player look command (picks up object your looking at)
 - /door - Player look command (Opens or Closes the door your looking at)
 - /cuff (player name) - arg command (cuffs based on player name)
 - /uncuff (player name) - arg command (uncuffs based on player name)
 - /fine (Player Name) (amount) (reason) - arg command (fines player the amount for a reason)
 - /PDAnnounce <type(Raid/Search)> <targetPlayerName> <targetRaidLocation> <time> - arg command 
 - /slimjim - Player look command
 - /vinspect - Player look command
 - /911  or /999 (Calltext) - Arg command
